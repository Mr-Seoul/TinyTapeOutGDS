This is my Tiny Tapeout submission. You can see more detailed documentation and the source code here: https://github.com/Mr-Seoul/TinyPong. 

The submission is an implementation of Pong, which should be able to be connected to a VGA adapter to display the game on a VGA monitor. The chip debounces inputs, so you only need to put a resistor (10k ohm at least) in series with the chip in order to be able to move the paddles.
